# Pirate Notebook - Full Stack MERN project
## Steps to download app<br/>
1) download the repo
2) Extract the files
3) Open terminal
4) Open Mongo DB and conne
5) cd react-pirate-notebook
6) npm install

## Steps to config mongo db and nodemon
1) Download and install mongoDB
2) npm i mongoose
3) npm i nodemon

## Steps to run app
1) npm run both
